2. Inheritance and Interfaces:- 
1. Single:- A single class is derived from a single base class.
2. Multiple:- A single class is derived from multiple base classes.
3. Hierarchial:- Multiple classes derived from a single base class.
4. Multilevel:- A single class is derived from another derived class.
5. Hybrid:- combination of multiple types of inheritance


3. Data Ecapsulation:- capsulating/binding of data(data members/varibles and member funcitons/methods)
